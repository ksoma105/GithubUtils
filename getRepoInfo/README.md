# GitHubStats
## What is this
You can get Github repository info.
- Stars 
- License
- Primary Language
- Created Date
- Commits(total)
- Commits(Half Year)
- Relases date(last:5)
- Versions(last:5)
- Contributors

## How to use
```bash
export GITHUB_TOKEN = YOUR_GITHUB_API_TOKEN
go run main.go
```
